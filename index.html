从 0 学习 Apache Paimon

即使你完全没有大数据基础，也能跟着这条路线，一步步理解 Paimon 是什么、为什么有用，以及怎么动手上手。

先说结论

Apache Paimon 是一个面向实时湖仓（Realtime Lakehouse）的开源数据湖表格式。它让流处理 + 批处理 + 数据湖存储更统一，尤其擅长主键更新、CDC 入湖、近实时查询。

这个网站适合谁

大数据零基础想转数据 / 数仓方向做实时数仓 / Flink想搞懂 Paimon vs Iceberg/Hudi/Delta

如果你看到 snapshot、LSM、CDC 这些词会发懵，别担心——这正是本教程要解决的问题。我们先打地基，再学 Paimon。
